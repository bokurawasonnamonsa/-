# Command Clock Privacy Policy

Last updated: June 12, 2026

Command Clock is a coordination timer app provided by TactNode Labs.

## Information We Collect

The current version of Command Clock does not collect personal information, account information, precise location, contacts, photos, messages, or payment information.

Command Clock stores timer settings, display preferences, custom labels, and share-code settings locally on your device so the app can remember your preferences.

## Network Use

The current version is designed to work without requiring an account or a dedicated server connection.

If future versions add online features, this policy will be updated before those features are released.

## Permissions

Command Clock may request the following permissions:

- Notifications, for timing alerts.
- Display over other apps, for the optional floating timer.
- Foreground service operation, to keep user-enabled timing features active while the app is running.

These permissions are used only to provide the app's timer, notification, and floating display features.

## Data Sharing

Command Clock does not sell personal data.

The current version does not share personal data with third parties.

## Children's Privacy

Command Clock is not designed to collect personal information from children.
