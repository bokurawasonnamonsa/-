## Full Description - en-US

Command Clock is a lightweight coordination timer for teams that need to act at the same moment.

Set your own timing value, the longest team timing value, a buffer, and a flow time. Command Clock calculates when each participant should press Start so the team can arrive or act together.

Key features:

- Synchronized countdown flow for coordinated team timing
- Floating countdown display for checking the timer while another app is open
- Share code support so a coordinator can pass timing settings to others
- 10 language UI support
- Simple controls for testing and adjusting team timing

Command Clock is designed for strategy, coordination, and event timing use cases. It does not require a dedicated server for early testing.

## Full Description - ja-JP

Command Clock は、チームで同じタイミングに動きたいときのための軽量な連携タイマーです。

自分の設定時間、チーム内で一番長い設定時間、猶予時間、進行中の残り時間を入力すると、各メンバーがいつスタートすれば同時に行動できるかを計算します。

主な機能:

- チームの同時行動を支える同期カウントダウン
- 他のアプリを開いていても確認しやすいフローティング表示
- 設定を共有しやすい共有コード機能
- 10言語対応のUI
- チームのタイミング調整に使いやすいシンプルな操作

Command Clock は、戦略、連携、イベント進行など、複数人でタイミングを合わせたい場面に向けたアプリです。初期テスト段階では専用サーバーなしで利用できます。
